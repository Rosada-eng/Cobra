# INSTRUÇÕES PARA RODAR O JOGO:

Instale as bibliotecas:
- pygame
- pytmx

Execute o arquivo v4_main.py

# - Como jogar

Há uma página de Instruções de como jogar dentro do próprio jogo. 
Na tela inicial, aperta 'i'. Após se informar, aperta 'i' novamente e, depois, aperte 'enter' para iniciar o jogo.

Você será uma cobrinha. Seu objetivo é devorar as 3 presas em cada mapa.
Coma as frutas para encher a Stamina. Quando sua stamina for suficiente, um simbolo aparecerá e você estará pronto para dar o bote.

Aperte 'X' para dar o bote quando se aproximar da presa.
Quando acertar o bote, a stamina será consumida. Não se preocupe, você receberá mais 30 seg a cada presa que você se alimentar.

Corra! Pois o tempo não para! Elimine as três presas de cada mapa. São dois mapas ao total.

Se falhar, você será direcionado para a tela de GAME OVER. 
Aperta 'ESPAÇO' para tentar novamente.

Ao completar com sucesso, você será destinado à tela de Vencedor.

Se quiser, você pode jogar novamente. Aperte ENTER e será direcionado para tela inicial novamente.

Se quiser SAIR, à qualquer momento, clice no X para fechar a janela.

Divirta-se!!
